# Entrega final · Plano A

1. Faça upload de `assets-upload/` para uma pasta exclusiva no HubSpot Files.
2. Execute `preparar-com-url.ps1` com a URL HTTPS dessa pasta.
3. Crie os três arquivos de `developer-files/` no Design Manager.
4. Use `evento-modelos-negocio-pronto.html` no lugar do template com placeholder.
5. Crie uma página nova em rascunho e siga o roteiro de QA.

O script não altera o arquivo-fonte; ele cria uma versão pronta ao lado do template.

## Conteúdo

- `developer-files/`: template HubL, CSS e JavaScript para o Design Manager.
- `assets-upload/`: imagens, logos, OG e favicon para o HubSpot Files.
- `METADADOS-DA-PAGINA.md`: SEO, Open Graph, formulário e pendências de publicação.
- `passo-a-passo-call.md`: roteiro operacional e QA.
- `CHECKSUMS-SHA256.txt`: integridade de todos os arquivos da entrega.
